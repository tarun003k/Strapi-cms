# Strapi Blog CMS

A full-stack blog website built using **Strapi (Headless CMS)** and **React**.

## Features

- Blog grid with images
- Categories, About, and Contact pages
- Responsive design (Tailwind CSS)
- Strapi backend for content management

## Getting Started

### 1. Clone the repository

git clone https://github.com/tarun003k/Strapi-cms.git
cd Strapi-cms

text

### 2. Install dependencies

- **Backend (Strapi):**
cd backend
npm install

text
- **Frontend (React):**
cd ../frontend
npm install

text

### 3. Start the project

- **Strapi backend:**
cd ../backend
npm run develop

text
- **React frontend:**
cd ../frontend
npm start

text

### 4. View the app

- Frontend: [http://localhost:3000](http://localhost:3000)
- Backend: [http://localhost:1337/admin](http://localhost:1337/admin)

## Deployment

- React frontend can be deployed on **Vercel**
- Strapi backend should be hosted using **Render**, **Railway**, **Heroku**, or your own VPS
